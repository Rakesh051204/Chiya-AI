# 🌱 Chiya AI

### Smart Food Saver & AI Food Assistant

Chiya AI is a smart food management platform that helps users track food inventory, reduce waste, and receive AI-powered recommendations on what to consume before items expire.

---

## 🚀 Features

### 📦 Food Inventory Management

* Add food items
* Track quantities
* Set expiry dates
* View inventory dashboard

### 🤖 AI Food Intelligence

* Smart food prioritization
* Expiry risk analysis
* Food waste reduction recommendations
* AI-powered consumption suggestions

### 📊 Dashboard Analytics

* Total food items
* Expired items count
* High-risk food alerts
* Inventory overview

### ☁️ Cloud Ready

* GitHub integration
* Streamlit deployment
* Supabase database support
* Multi-user SaaS architecture (coming soon)

---

## ▶️ Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Start the application:

```bash
python -m streamlit run app.py
```
